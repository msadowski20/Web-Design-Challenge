Project Summary

In this project we looked at weather patterns for locations around the world and how they change as you get closer to the equator. We pulled weather data for over 500 cities from the OpenWeatherMap API.

We used Python and Matplotlib to perform the anlysis and looked at several weather factors and compared each to the latitude of the city. The factors that we analyzed were temperature, humidity, cloudiness and wind speed. This site is a visualization of all the original data as well as the graphs created as part of the analysis. You will find a graph of each weather factor vs. latitude and a comparison of all graphs, along with the description of any correlations or trends that were discovered.
